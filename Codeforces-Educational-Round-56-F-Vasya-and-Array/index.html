[Codeforces Educational Round 56] F Vasya and Array

发表于2018-12-25更新于2020-08-06

题目大意

定义一个数列是好的，当且仅当它的任何长为 lenlenlen 的连续子序列包含至少两种数字。现给出一个长为 nnn 的数列，每个位置是 1∼k1 \sim k1∼k 中的数或 −1-1−1，要求替换所有的 −1-1−1 为 1∼k1 \sim k1∼k 中的数，使得该数列为好数列，求替换的方案数。

1≤len≤n≤100,0001 \leq len \leq n \leq 100,0001≤len≤n≤100,000

1≤k≤1001 \leq k \leq 1001≤k≤100

题解

考虑 DP，f(i,j)f(i, j)f(i,j) 表示已经考虑了前 iii 个数，最后一个是 jjj 的答案，则在不考虑要求的情况下，有：

f(i,j)=∑j′=1kf(i−1,j′)f(i, j) = \sum_{j' = 1}^{k} f(i - 1, j') f(i,j)=j′=1∑k​f(i−1,j′)

然后考虑减去不合法的情况，由于更早的不合法情况已经考虑过了，只需考虑 [i−len+1,i][i - len + 1, i][i−len+1,i] 这个区间（如果 i−len+1>0i - len + 1 > 0i−len+1>0），它们只有都是 jjj 的情况下才不合法，这需要原数列中该区间内只有 jjj 和 −1-1−1，此时减去不合法的情况数，也就是前 i−leni - leni−len 个数的答案，此时如果第 i−leni - leni−len 个数也为 jjj，那么它在计算 f(i−1,j)f(i - 1, j)f(i−1,j) 时就被考虑了，故减去 ∑j′=1kf(i−len,j′)−f(i−len,j)\sum_{j' = 1}^{k} f(i - len, j') - f(i - len, j)∑j′=1k​f(i−len,j′)−f(i−len,j) 。
