Contest 10 - A - Bnumbers

题目大意

一个数 kkk 在 aaa 进制下的表示为 xnxn−1…x1x0x_nx_{n-1} \dots x_1x_0xn​xn−1​…x1​x0​。定义 kkk 是 aaa 进制下的一个 B 数，当且仅当：

∑i=0nxi2=k\sum_{i = 0}^{n} x_i^2 = k i=0∑n​xi2​=k

求 aaa 进制下共有多少个 B 数。

2≤N≤1,0002 \leq N \leq 1,0002≤N≤1,000

2≤a≤1062 \leq a \leq 10^62≤a≤106

∑a≤108\sum a \leq 10^8∑a≤108

时间限制：2s2 \text{s}2s，内存限制：64MB64 \text{MB}64MB。

题解

可以证明，B 数要么是 111，要么是一个 aaa 进制下的两位数。写出对应的不定方程，O(a)O(a)O(a) 枚举求解另一变量并判断计数即可。

给定一个长为 nnn 的序列 {ai}\{a_i\}{ai​}，求至少删除多少的数后，序列的最长上升子序列的长度会减少。

0≤n≤1,0000 \leq n \leq 1,0000≤n≤1,000

0≤ai≤1090 \leq a_i \leq 10^90≤ai​≤109

1≤T≤51 \leq T \leq 51≤T≤5

时间限制：1s1 \text{s}1s，内存限制：64MB64 \text{MB}64MB。

考虑计算 LIS 的 DP 过程，如果 f(i)=f(j)+1f(i) = f(j) + 1f(i)=f(j)+1，则建一单向条边 (j,i)(j, i)(j,i) ，拆点跑最小割即为答案。

Contest 11 - A - Haibara and Bit Strings

题目大意

长为 n=10mn = 10^mn=10m 的 010101 串中，010101 作为子串出现了两次的有多少个，答案对 1,000,000,0071,000,000,0071,000,000,007 取模。

1≤m≤1071 \leq m \leq 10^71≤m≤107

1≤T≤10,0001 \leq T \leq 10,0001≤T≤10,000

时间限制：1s1 \text{s}1s，内存限制：64MB64 \text{MB}64MB。

题解

满足条件的串都长这样：

1…10…01…10…01…10…01\dots10\dots01\dots10\dots01\dots10\dots0 1…10…01…10…01…10…0

问题转化为 a+b+c+d+e+f=na + b + c + d + e + f = na+b+c+d+e+f=n 的非负整数解的个数，由隔板法知答案为 (n+15)\binom{n + 1}{5}(5n+1​)。

Contest 11 - E - JSB’s Happy Life 1

题目大意

有 nnn 个数 {ai}\{a_i\}{ai​}，定义其一个子集的价值为集合中 aia_iai​ 的和，求其所有非空子集中，价值第 kkk 小的价值。

1≤n,k,ai≤100,0001 \leq n, k, a_i \leq 100,0001≤n,k,ai​≤100,000

1≤T≤201 \leq T \leq 201≤T≤20

时间限制：1s1 \text{s}1s，内存限制：64MB64 \text{MB}64MB。

题解

数的顺序对答案无影响，故可以现按升序对其排序。对于每一个状态，其后继都有两种选择：加上下一个；或先删去当前的，再加上下一个。用一个优先队列储存，先放入所有元素，每次弹出一个并放入其两个后继至完成 k−1k - 1k−1 次操作，此时优先队列的队首即为答案。

Contest 12 - B - zx2018 and Study

题目大意

求方程

∑i=1ai=2∑i=1b−1+b\sum_{i = 1}^{a}i = 2\sum_{i = 1}^{b - 1} + b i=1∑a​i=2i=1∑b−1​+b

从小到大的第 nnn 组解。

1≤n≤10121 \leq n \leq 10^{12}1≤n≤1012

1≤T≤1,0001 \leq T \leq 1,0001≤T≤1,000

时间限制：2s2 \text{s}2s，内存限制：64MB64 \text{MB}64MB。

题解

方程可变形为：

(2a+1)2−8b2=1(2a + 1)^2 - 8b^2 = 1 (2a+1)2−8b2=1

令 u=2a+1,v=bu = 2a + 1, v = bu=2a+1,v=b，则方程为一个佩尔方程。由相关理论知：

\begin{align} u_n &= u_{n - 1}u_0 + v_{n - 1} + 8 \\ v_n &= v_{n - 1}u_0 + u_{n - 1}v_0 \end{align}

复原 a,ba, ba,b ，用矩阵乘法快速幂计算即可。
