[HAOI 2011] Problem b

发表于2017-04-04更新于2020-08-06

题目大意

对于给出的 nnn 个询问，每次求有多少个数对 (x,y)(x, y)(x,y)，满足 a⩽x⩽ba \leqslant x \leqslant ba⩽x⩽b，c⩽y⩽dc \leqslant y \leqslant dc⩽y⩽d，且gcd(x,y)=kgcd(x, y) = kgcd(x,y)=k。

n, a, b, c, d, k⩽50,000n, \ a, \ b, \ c, \ d, \ k \leqslant 50,000n, a, b, c, d, k⩽50,000

题目链接

【HAOI 2011】problem b - Luogu 2522

题解

莫比乌斯反演裸题（本来还有一个更裸，但被权限了。。。）。

首先，通过容斥，我们可以把问题化为求满足 x⩽nx \leqslant nx⩽n，y⩽my \leqslant my⩽m，且 gcd(x,y)=kgcd(x, y) = kgcd(x,y)=k 的数对个数。即：

∑x=1n∑y=1m[gcd(x, y)=k]\sum_{x = 1}^{n} \sum_{y = 1}^{m} [gcd(x, \ y) = k] x=1∑n​y=1∑m​[gcd(x, y)=k]

我们这么对式子进行变换：

\begin{align} \sum_{x = 1}^{n} \sum_{y = 1}^{m} [gcd(x, \ y) = k] & = \sum_{x = 1}^{n'} \sum_{y = 1}^{m'} [gcd(x, \ y) = 1] \quad (n' = \lfloor \frac{n}k \rfloor, \ m' = \lfloor \frac{m}k \rfloor) \\ & = \sum_{x = 1}^{n'} \sum_{y = 1}^{m'} \sum_{d | gcd(x, y)} \mu(d) \\ & = \sum_{i = 1}^{min(n', m')} \mu(i) \lfloor \frac{n'}i \rfloor \lfloor \frac{m'}i \rfloor \end{align}

其中第二行的式子来源于 μ ∗ 1=e\mu \ * \ 1 = eμ ∗ 1=e，μ\muμ 为莫比乌斯函数，111 为常函数（函数值为 111），eee 为元函数（除 x=1x = 1x=1 处的值为 111 以外函数值为 000），∗*∗ 为狄利克雷卷积。

注意到，在最后的式子中，⌊n′i⌋\lfloor \frac{n'}i \rfloor⌊in′​⌋ 只有 2n2\sqrt{n}2n​ 个值（⌊m′i⌋\lfloor \frac{m'}i \rfloor⌊im′​⌋ 同），所以我们可以预处理出莫比乌斯函数的前缀和，分块进行计算。
