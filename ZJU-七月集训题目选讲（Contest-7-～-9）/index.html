Contest 7 - B - Die Wacht am Rhein

题目大意

给定一张 NNN 个点、MMM 条边的森林，每个点有一个权值 aia_iai​。现要增加若干条边，使得图连通，且满足每个顶点最多连接了一条新边。定义新增一条边 (i,j)(i, j)(i,j) 的代价为 ai+aja_i + a_jai​+aj​，求最小代价，或判断它不可能实现。

1≤N≤100,0001 \leq N \leq 100,0001≤N≤100,000

0≤M≤N−10 \leq M \leq N - 10≤M≤N−1

1≤ai≤1091 \leq a_i \leq 10^91≤ai​≤109

∑N≤2,500,000\sum N \leq 2,500,000∑N≤2,500,000

时间限制：1s1 \text{s}1s，内存限制：256MB256 \text{MB}256MB。

题解

如果原图只有一个连通块，则答案为 000；否则每个连通块内至少有一个点连接了新边，我们选择连通块内权值最小的点加入答案。

由于输入是一个森林，我们可以知道原图有 N−MN - MN−M 个连通块，连接它们需要 N−M−1N - M - 1N−M−1 条边，即 2(N−M−1)2(N - M - 1)2(N−M−1) 个点。在每个连通块加入一个点后，仍需要 N−M−2N - M - 2N−M−2 个点。我们发现，连边 (p,q)(p, q)(p,q) 和 (s,t)(s, t)(s,t) 所需的代价等于连边 (p,t)(p, t)(p,t) 和 (s,q)(s, q)(s,q)，即代价与具体连边方式无关。我们把剩下的点的权值按升序排序，取最小的 N−M−2N - M - 2N−M−2 个点加入答案即可。

Contest 7 - C - Yet another Chess Puzzle

题目大意

给定一个 n×nn \times nn×n 的网格，保证 n+1n + 1n+1 为质数。在网格内放入 nnn 个棋子，使它们满足：

  • 任意两个棋子不在同一行、列；
  • 不存在两个点对 (A,B)(A, B)(A,B) 和 (C,D)(C, D)(C,D) （点不完全相同即算数），满足 ∣xA−xB∣=∣xC−xD∣|x_A - x_B| = |x_C - x_D|∣xA​−xB​∣=∣xC​−xD​∣ 且 ∣yA−yB∣=∣yC−yD∣|y_A - y_B| = |y_C - y_D|∣yA​−yB​∣=∣yC​−yD​∣。

输出一个满足以上条件的摆放方式。

2≤n≤5,000 (n+1 is a prime number)2 \leq n \leq 5,000 ~ (n + 1 \text{ is a prime number})2≤n≤5,000 (n+1 is a prime number)

1≤T≤1,0001 \leq T \leq 1,0001≤T≤1,000

时间限制：1s1 \text{s}1s，内存限制：256MB256 \text{MB}256MB。

题解

我们可以求出 n+1n + 1n+1 的一个原根 ggg，则 (0,g0),(1,g1),…,(n−1,gn−1)(0, g^0), (1, g^1), \dots, (n - 1, g^{n - 1})(0,g0),(1,g1),…,(n−1,gn−1) 即是答案。

第一个限制显然满足了，下证其满足第二个限制：

假设存在两个点对不满足限制二，则有：

\begin{align} g^{x_2} - g^{x_1} &\equiv g^{x_4} - g^{x_3} (\bmod n + 1) \\ g^{x_1 + dx} - g^{x_1} &\equiv g^{x_3 + dx} - g^{x_3} (\bmod n + 1) \\ g^{x_1}(g^{dx} - 1) &\equiv g^{x_3}(g^{dx} - 1) (\bmod n + 1) \\ g^{x_1} &\equiv g^{x_3} (\bmod n + 1) \end{align}

与原根的性质不符。

Contest 8 - C - Bulbasaur and Prime Game

题目大意

有 nnn 个数 {ai}\{a_i\}{ai​} 。两个人进行游戏：当上一个被取走的数为 axa_xax​ 时，该行动的一方可以从这 nnn 个数中选取一个数 aya_yay​，满足 ax+aya_x + a_yax​+ay​ 是一个质数；起初 ax=0a_x = 0ax​=0；一方不能行动时另一方获胜。求必胜方。

1≤n≤1,0001 \leq n \leq 1,0001≤n≤1,000

2≤ai≤1092 \leq a_i \leq 10^92≤ai​≤109

∑n≤5,000\sum n \leq 5,000∑n≤5,000

时间限制：1s1 \text{s}1s，内存限制：64MB64 \text{MB}64MB。

题解

对于每一对满足 ax+aya_x + a_yax​+ay​ 为质数的数对，建一条边 (x,y)(x, y)(x,y)，则可以得到一个二分图。则问题转换为一个二分图游戏。建图跑一遍网络流/匈牙利并判断即可。

对于判质数，我们可以预处理一定范围内的质数，或用 Pollard’s Rho。

Contest 8 - F - Totodile and Graph Theory

题目大意

给定一张 nnn 个点、mmm 条边的图和一个数 KKK。要求给图进行 KKK 染色，或输出一条长为 KKK 的简单路径，或判断以上两问均无法实现。

2≤n≤100,0002 \leq n \leq 100,0002≤n≤100,000

0≤m≤min⁡(n(n−1)2,100,000)0 \leq m \leq \min(\frac{n(n - 1)}{2}, 100,000)0≤m≤min(2n(n−1)​,100,000)

2≤K≤n2 \leq K \leq n2≤K≤n

∑n,∑m≤1,000,000\sum n, \sum m \leq 1,000,000∑n,∑m≤1,000,000

时间限制：1s1 \text{s}1s，内存限制：64MB64 \text{MB}64MB。

题解

随便选取一个点作为起点进行 dfs。若最深深度大于等于 KKK，则一定存在一条长度为 KKK 的路径；否则为深度相同的点染相同的色、不同的点染不同的色，由于 dfs 树的性质，不存在一条非树边连接两个深度相同的点，即一定可以 KKK 染色。

Contest 8 - G - Totodile and Another Data Structure Problem?

题目大意

给定一个长为 nnn 的序列 {ai} (ai∈{1,2})\{a_i\} ~ (a_i \in \{1, 2\}){ai​} (ai​∈{1,2})。定义一个数 ggg 是好的，当且仅当存在区间 [l,r][l, r][l,r] 满足 ∑i=lrai=g\sum_{i = l}^{r} a_i = g∑i=lr​ai​=g。求所有不超过 kkk 的好数的异或和。

1≤n≤100,0001 \leq n \leq 100,0001≤n≤100,000

1≤k≤1091 \leq k \leq 10^91≤k≤109

∑n≤1,000,000\sum n \leq 1,000,000∑n≤1,000,000

时间限制：1s1 \text{s}1s，内存限制：64MB64 \text{MB}64MB。

题解

可以证明，一个至少有一端为 111 的序列的好数为 [1,∑ai][1, \sum a_i][1,∑ai​] 内的所有整数（称这种序列为好序列）。对于两端均为 222 的序列，可以拆成一段连续的 222 和一个好序列 {bi}\{b_i\}{bi​}。枚举 222 的个数 kkk，则 2k+∑bi2k + \sum b_i2k+∑bi​ 为好数。最后直接计算异或和即可。

Contest 9 - A - LYK and Heltion’s gifts

题目大意

用两种颜色染一个有 nnn 个珠子的首饰，要求不能有相邻的黑色珠子，求可能的染色方案数，旋转相同视为同一种，答案对 1,000,000,0071,000,000,0071,000,000,007 取模。

2≤n≤1062 \leq n \leq 10^62≤n≤106

1≤T≤1,0001 \leq T \leq 1,0001≤T≤1,000

时间限制：2s2 \text{s}2s，内存限制：512MB512 \text{MB}512MB。

题解

旋转同构用 Burnside 引理处理。对于不考虑旋转时的答案（记为 f(n)f(n)f(n)）可以发现是：

\begin{align} f(2) &= 3 \\ f(3) &= 4 \\ f(n) &= f(n - 1) + f(n - 2) \end{align}

1≤N≤231−11 \leq N \leq 2^{31} - 11≤N≤231−1

1≤M≤100,0001 \leq M \leq 100,0001≤M≤100,000
