[Codeforces Educational Round 56] G Multidimensional Queries

发表于2018-12-24更新于2020-08-06

题目大意

{ai}\{a_i\}{ai​} 是 kkk 维空间内的 nnn 个点，有 qqq 次操作，每次操作为修改一个点的坐标，或查询下标区间在 [l,r][l, r][l,r] 内的点的最大两点间曼哈顿距离。

1≤n,q≤200,0001 \leq n, q \leq 200,0001≤n,q≤200,000

1≤k≤51 \leq k \leq 51≤k≤5

−106≤ai,j≤106-10^6 \leq a_{i, j} \leq 10^6−106≤ai,j​≤106

题目链接

Codeforces Educational Round 56 - G

题解

jjj、kkk 两点间的曼哈顿距离可表示为：

∑i=1kciaj,i−∑i=1kciak,i\sum_{i = 1}^{k} c_i a_{j, i} - \sum_{i = 1}^{k} c_i a_{k, i} i=1∑k​ci​aj,i​−i=1∑k​ci​ak,i​

其中 cic_ici​ 为一个 {1,−1}\{1, -1\}{1,−1} 的系数，且可以直接证明，对任何 cic_ici​，曼哈顿距离为其中最大的一个。由于 kkk 只有 555，可以考虑每举所有 2k2^k2k 种 cic_ici​，询问即区间内求 ∑i=1kciaj,i\sum_{i = 1}^{k} c_i a_{j, i}∑i=1k​ci​aj,i​ 的最值，修改即单点修改，用 2k2^k2k 棵线段树可以做到 O((n+q)2klog⁡n)O((n + q) 2^k \log n)O((n+q)2klogn)。由于每次操作的点或区间对所有线段树均相同，故可用一棵线段树维护，复杂度降为 O((n + q)2^k + q \log n)​ ，复杂度不会改变，但常数会变小很多，因为这样对内存访问更加友好；同时，由于 ci=cc_i = cci​=c 时的最大值就是 ci=2k−1−cc_i = 2^k - 1 - cci​=2k−1−c 时的最小值，故只用维护一种最值。
