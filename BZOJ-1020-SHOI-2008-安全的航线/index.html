[SHOI 2008] 安全的航线

发表于2017-05-09更新于2020-08-06

题目大意

航线用一段有 nnn 个点的折线表示，ccc 块陆地用不相交的多边形表示，定义「孤地点」为距离最近的陆地最远的地方，求孤地距离。

2⩽n⩽202 \leqslant n \leqslant 202⩽n⩽20

1⩽c⩽201 \leqslant c \leqslant 201⩽c⩽20

∣x∣,  ∣y∣⩽10,000|x|, \; |y| \leqslant 10,000∣x∣,∣y∣⩽10,000

题目链接

【SHOI 2008】安全的航线 - Luogu 4286

题解

%%%%% 莫涛神犇：莫涛《迭代思想的应用》

%%%%% ydc 神犇：BZOJ 1020 | ydc的博客

把折线的每个线段放入队列，用 nnn 个点更新一波答案。

每取出一条线段，记其两个端点到陆地的最近点为 p1p_1p1​ 、p2p_2p2​，二分找到线段上的点 ppp 使得 dist(p,p1)=dist(p,p2)dist(p, p _1) = dist(p, p_2)dist(p,p1​)=dist(p,p2​) ，记为 rrr ，这是当前线段能更新出的最大答案（而且可能去取不到），若有 r⩽ansr \leqslant ansr⩽ans 则直接删除该线段，否则从 ppp 处切断线段并放入队列。反复以上操作直至队列为空。

一些细节需要注意，比如要写成 r <= ans + 0.05 就不会无法结束算法。

自己的方法名好长。。。（ Java 的既视感？）
