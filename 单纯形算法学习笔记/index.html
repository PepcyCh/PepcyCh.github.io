单纯形算法学习笔记

发表于2017-04-04更新于2020-08-06

算法介绍

从 Po 姐那里看到的 wyfcyx 大佬的《线性规划与单纯形算法》，推荐去看一看。

以下内容来自于那个课件。

规定一个线性规划的标准型：

在条件 Ax⩽bAx \leqslant bAx⩽b 下，最大化 cTxc^T xcTx。

其中各字符均为矩阵，其意义如下（nnn 为变量数，mmm 为限制条件数）：

AAA：限制条件的系数矩阵，大小为 m×nm \times nm×n。

xxx：变量矩阵，大小为 n×1n \times 1n×1。

bbb：限制条件的常数矩阵，大小为 m×1m \times 1m×1。

ccc：目标函数的系数矩阵，大小为 n×1n \times 1n×1。

下面考虑如何把不是标准型的线性规划转化为标准型。

  • 目标函数最小化：所有系数取反，转化为最大化，最后对答案取反。
  • 限制条件为大于等于：系数取反。
  • 限制条件为等于：拆成大于等于、小于等于两个限制。
  • 变量限制条件为 x⩾ax \geqslant ax⩾a：转化为x′⩾0, x′=x−ax' \geqslant 0, \ x' = x - ax′⩾0, x′=x−a。
  • 变量限制条件为 x⩽ax \leqslant ax⩽a：转化为x′⩾0, x′=a−xx' \geqslant 0, \ x' = a- xx′⩾0, x′=a−x。
  • 变量无限制：转化为 x′⩾0, x′′⩾0, x=x′−x′′x' \geqslant 0, \ x'' \geqslant 0, \ x = x' - x''x′⩾0, x′′⩾0, x=x′−x′′。

线性规划的松弛型：

给第i个变量添加辅助变量 xn+ix_{n + i}xn+i​，把不等号转化为等号。

xn+i+∑j=1nAi,jxj=bi, xn+i⩾0x_{n + i} + \sum_{j = 1}^{n} A_{i, j}x_j = b_i, \ x_{n + i} \geqslant 0 xn+i​+j=1∑n​Ai,j​xj​=bi​, xn+i​⩾0

现在，定义一些名称：

基变量：把标准型转化为松弛型的辅助变量，有 mmm 个，其集合记为 BBB。

原变量：原来的变量，有 nnn 个，其集合记为 NNN。（这个在那个课件里叫“非基变量”）

可行解：使所有限制满足的一组原变量。

最优解：使目标函数最优的一组可行解。

算法过程：

先假定所有的原变量为 000 为一组可行解，考虑目标函数中某个原变量，若其系数大于 000 ，那我们增大那个原变量会使目标函数值增大。

若所有原变量的系数均小于等于 000，则当前解为最优解，目标函数值的最大值即为目标函数的常数。

若我们已经找到了一个系数大于 000 的原变量，假设是第 eee 个，记为 xNex_{N_e}xNe​​。

关于它的限制有：

xBi+∑j=1nAi,jxNj=bix_{B_i} + \sum_{j = 1}^{n} A_{i, j} x_{N_j} = b_i xBi​​+j=1∑n​Ai,j​xNj​​=bi​

若 Ai,e>0A_{i, e} > 0Ai,e​>0，则有：

Ai,e⩽biAi,eA_{i, e} \leqslant \frac{b_i}{A_{i, e}} Ai,e​⩽Ai,e​bi​​

记找到的最紧的限制条件为第 lll 个，其对应基变量为 xBlx_{B_l}xBl​​。

之后进行转轴操作，即交换找到的基变量与原变量，同时进行换元之类的操作使得新基变量的系数为 111。操作完毕后，我们发现，目标函数的常数项增大了。

注意：选择原变量时，应选择编号最小的，否则容易被特殊数据卡出死循环。

再说一个对偶原理，可以用于转化线性规划到标准型（很常用的样子），其内容如下：

有两个线性规划：

  • 在条件 Ax⩽bAx \leqslant bAx⩽b下，最大化 cTxc^T xcTx。

  • 在条件 ATy⩾cA^Ty \geqslant cATy⩾c下，最小化 bTyb^T ybTy。

它们的最优解相同。
