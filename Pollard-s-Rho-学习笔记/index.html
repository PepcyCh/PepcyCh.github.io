Pollard's Rho 学习笔记

发表于2017-05-05更新于2020-08-06

Pollard’s Rho 算法是用于分解质因数的算法。

算法介绍与过程

用 Miller-Rabin 算法判断当前要分解的数是否为质数，若是，在对应位上指数加一，直接返回。

若不是，找到一个数 ppp 是它为 nnn 的约数，然后对 n/pn / pn/p 、ppp 递归求解。

假设我们现在要找 nnn 的一个约数 ppp，先让 p=np = np=n ，之后进行计算。

先随机一个数 xxx ，并令 y=xy = xy=x（wiki 上是令 x=y=2x = y = 2x=y=2 ），按以下方式进行判断：

让 x=g(x)x = g(x)x=g(x) ，计算 d=gcd(∣x−y∣,p)d = gcd(|x - y|, p)d=gcd(∣x−y∣,p) ，如果 ddd 是 111 则让 y=g(g(y))y = g(g(y))y=g(g(y)) （实现时并没有每次都改变 yyy，而是在计算了 222、444、888、161616 …… 次后让 y=xy = xy=x）并继续重复该算法；若 d=nd = nd=n，则判断失败（一般返回 ppp 让重新寻找）；否则返回 ddd 作为约数。

其中 g(x)g(x)g(x) 一般为 g(x)=(x2+c) mod pg(x) = (x^2 + c) \bmod pg(x)=(x2+c)modp ，ccc 为常数，一般为 111 ，在重新计算时改变常数。

如果在寻找时出现了 x=yx = yx=y ，则意味着出现了环，返回失败并让重新寻找。xxx、yyy 形成的环一般为 ρ\rhoρ 型，这就是算法名字的由来。另外，取 x=g(x),  y=g(g(y))x = g(x), \; y = g(g(y))x=g(x),y=g(g(y)) 也是为了能够判圈（就像 Floyd 判圈法）。
